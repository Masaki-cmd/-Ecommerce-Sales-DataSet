# -Ecommerce-Sales-DataSet
## 🔷 README: E-commerce Sales Data Analysis Report

### プロジェクト概要 (Project Overview)

本プロジェクトは、提供されたEコマース販売データを分析し、特に「収益の最大化」というKGIを達成するための「意思決定に直結する構造化インサイト」を生成することを目的としています。データクリーニング、探索的データ分析（EDA）を通じてビジネスパフォーマンスの全体像を把握し、特に低パフォーマンスが確認された「Books」カテゴリに対し、ROAS（広告費用対効果）改善のための詳細分析と具体的な施策提案を行いました。

### データ概要 (Data Overview)

*   **データソース**: `ecommerce_sales_data.csv` ( `archive (1).zip` から抽出)
*   **初期データサイズ**: 1000行, 9列
*   **クリーン後データサイズ**: 734行, 8列 ( `Year`, `Month` を `Date` に統合後、`Date`, `Category`, `Region` で集計)
*   **列**: `Date`, `Category`, `Region`, `Units_Sold`, `Revenue`, `Profit`, `Customer_Rating`, `Ad_Spend`
*   **データ品質スコア**: 94/100 (主な理由: 外れ値の存在)
*   **欠損値**: なし
*   **重複行**: なし (集計により対応)

### 分析手法 (Analysis Methods)

1.  **Data Understanding**: データ構造、型、欠損値、重複、主キー候補、分析粒度の特定。
2.  **Business Understanding**: KGI「収益の最大化」、KPI（平均販売数、平均利益、平均顧客評価、広告費用対効果）の定義とKPIツリーの構築。
3.  **Data Cleaning**: 
    *   `Year`と`Month`を結合し`Date`列（datetime型）を作成。
    *   `Category`と`Region`をカテゴリ型に変換。
    *   `Date`, `Category`, `Region`の組み合わせで非ユニークなレコードを集計し、データ粒度を統一。
    *   数値列（`Units_Sold`, `Revenue`, `Profit`, `Ad_Spend`）における外れ値をIQR法で検出（除去はせず報告のみ）。
    *   業務ロジックに基づく異常値は検出されず。
4.  **EDA (Adaptive EDA)**:
    *   全体: 統計的要約、数値変数の分布、時系列トレンド、カテゴリ別・地域別の平均パフォーマンス比較、数値変数間の相関分析。
    *   Booksカテゴリ詳細分析: 統計的要約、時系列トレンド、地域別パフォーマンス、ROAS算出とROASに対する影響要因（`Units_Sold`, `Profit`, `Customer_Rating`）の相関分析。

### 主要インサイト (Key Insights)

*   **全体パフォーマンス**: `Home Decor`と`Groceries`が平均収益・利益でトップのカテゴリ。`North`と`East`がトップリージョン。`Units_Sold`, `Revenue`, `Profit`間には強い正の相関。`Ad_Spend`と`Units_Sold`には中程度の相関。`Customer_Rating`は他の指標との相関が弱い。
*   **Booksカテゴリの課題**: 他のカテゴリと比較して平均収益、利益、ROASが低い。時系列トレンドは成長が停滞傾向。特に`West`および`South`地域でのROASが低い。
    *   **ROAS分析**: BooksカテゴリにおけるROASの平均は約10.68。ROASと`Profit`には0.48、`Units_Sold`には0.19の相関。`Customer_Rating`との相関は非常に弱い（-0.09）。地域別では`West`（約9.50）が最もROASが低く、`South`（約10.66）がそれに続く。

### 施策 (Recommendations)

**BooksカテゴリにおけるROAS改善と低ROAS地域（West/South）への具体的戦略**:

1.  **予算再配分の検討**: ROASの高い地域/カテゴリから、BooksカテゴリのWest/South地域へ全体の広告予算の5-10%をテスト予算として段階的に再配分。
2.  **地域特化型広告戦略**:
    *   **ターゲット層の再定義とローカライズ**: West/South地域の読者層のデモグラフィック、興味関心、読書習慣を詳細に調査し、ターゲティング精度を向上。
    *   **クリエイティブとメッセージの最適化**: 各地域の文化的背景、言語的ニュアンスを考慮した広告クリエイティブやメッセージを作成。
    *   **広告チャネルの最適化**: 各地域で高いリーチを持つオンラインプラットフォームやローカルメディアを活用。
    *   **ランディングページ（LP）の最適化**: 地域限定プロモーションや推薦書籍リストなど、地域特性に合わせたLPコンテンツを提供。
3.  **パフォーマンスモニタリングと改善サイクル**: West/South地域におけるROAS, CTR, CVR, AOVを主要KPIとして継続的に追跡。A/Bテストを頻繁に実施し、月次で戦略と予算配分を見直し。

### ROI分析（ある場合） (ROI Analysis (if available))

現在利用可能なデータには、個別のコスト情報や製品単価が含まれていないため、正確なROI分析は実施できませんでした。ROI分析を実行するためには、以下の情報が必要です。

*   製品ごとの製造コスト、仕入れコスト
*   物流コスト、人件費などの変動費用・固定費用

これらのデータが提供された場合、収益性分析を深掘りし、真の投資対効果を算出することが可能となります。

### 限界 (Limitations)

*   **ROI分析の欠如**: コストデータがないため、真の投資収益率を算出し、予算配分を最適化する上での制約があります。
*   **因果関係の特定**: 相関分析は因果関係を示すものではないため、施策の導入に際してはA/Bテストなどによる効果検証が不可欠です。
*   **外れ値の影響**: 外れ値は検出されましたが除去されていないため、一部の統計的要約や平均値に影響を与えている可能性があります。
*   **データ期間**: 提供されたデータは特定の期間のものであるため、長期的なトレンドや季節性の変化をより正確に捉えるには、より長期間のデータが必要となる場合があります。
